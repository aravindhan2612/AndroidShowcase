## NdKCounterApp

---

## Architecture

This repository follows the MVVM Clean Architecture pattern:

```
 Presentation ( UI, ViewModel)
      |
      |
    Domain  (Repo Interface, Models, Usecases)
      |
      |
     Data  (Repo Impl) 
```
